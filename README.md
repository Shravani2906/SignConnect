# 🤟 SignConnect

SignConnect is an interactive sign language learning web application designed to help users learn and practice sign language in a simple and engaging way.

## 🚀 Features

- Learn alphabets, numbers, and vocabulary
- Practice mode with structured levels
- Daily quiz system
- Streak tracking system
- User profile with image upload
- Dark mode support
- Fully responsive (mobile + desktop)

## 🛠 Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- LocalStorage
